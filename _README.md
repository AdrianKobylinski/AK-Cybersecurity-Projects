# AK-Cybersecurity-Projects

A collection of hands-on cybersecurity labs, investigations, and projects focused on Security Operations Center (SOC) workflows, threat detection, incident response, threat hunting, and defensive security.

This repository serves as my cybersecurity portfolio and learning journal, documenting my progression from foundational security concepts to practical blue team skills.

## Current Background

* ISC2 Certified in Cybersecurity (CC)
* Completed the TryHackMe SOC Level 1 learning path
* Building hands-on experience through home labs, log analysis, threat detection exercises, and SOC-focused projects

* Develop practical SOC Analyst skills
* Gain experience with security monitoring and investigation workflows
* Practice threat detection using real-world tools and telemetry
* Learn and apply MITRE ATT&CK techniques
* Document findings, lessons learned, and investigation methodologies
* Build a portfolio of projects demonstrating defensive security knowledge


This repository will continue to grow as I expand my cybersecurity knowledge and complete additional labs, investigations, and defensive security projects.
